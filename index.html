behavioral detection with computer vision

For six weeks in the summer of 2022 I worked in a team of two on using computer vision to detect certain behaviors from camera footage of poultry. We used YOLOv5 and DeepSORT for individual object detection and tracking, and a combination of heuristics and a convolutional LSTM network to build binary classifiers. Source code is not open for this project, unfortunately.

mcmc implementation using r

One of my university courses had a project whose main objective was to implement an MCMC sampler "by hand", with the statistical programming language R. In addition, we implemented an option to use "blocking" of Gibbs- and MH-steps to improve preformance.

estimating hamiltonial energies using neural networks and numerical optimizers

A project in the course of numerical mathematics, where we first implement an algorithm with a combination of Adam-optimizer and SGD to produce a function whose gradient is computed. Using numerical integrators such as Symplectic Euler, we can compute properies like the total energy of an unknown hamiltonial system.
